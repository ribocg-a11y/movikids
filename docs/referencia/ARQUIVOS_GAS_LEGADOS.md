# Arquivos GAS legados no repositório

Os arquivos `MOVIKIDS_Code_v1.5.*.gs` antigos (exceto o canônico abaixo) são **histórico de versões**. Muitos ainda declaram `DEPLOY_ID = AKfycbzc...` (URL morta).

**Produção usa apenas:**

- `MOVIKIDS_Code_v1.5.32_AUTH_OPERADORES_SOBRE_v1.5.31.gs` → copiado para `gas/Code.gs` via `scripts/sync-gas-to-clasp.ps1`

Não implantar nem fazer `clasp push` a partir de arquivos legados.
